# homebrew-godocs

Homebrew tap for [godocs](https://github.com/Theholetruth-org/Hole-GoDocs) — the evidence-ETL CLI for FOIA disclosure bundle preparation.

## Install

```bash
brew tap theholetruth-org/godocs
brew install godocs
```

Then verify:

```bash
godocs --version
```

## Upgrade

```bash
brew upgrade godocs
```

## What's installed

- **`godocs`** — main CLI (split, bates, rag, filing, toc, render, …)
- **`godocs-pdf-merge`** — PDF merge/validate utility

## Releases

Pre-built macOS binaries (arm64 + amd64) are mirrored to [Theholetruth-org/Hole-GoDocs](https://github.com/Theholetruth-org/Hole-GoDocs/releases) on each stable release. The formula here is updated automatically by the release workflow in the source repo.
